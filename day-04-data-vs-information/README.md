# Day 4: Data vs. Information

## Simple Explanation

Data is raw facts.

Information is data that has been organized, interpreted, or given context so it becomes meaningful.

Example: 42 failed login attempts is data. An unusual spike in failed logins that may indicate a password attack is information.

## Why It Matters At Work

Every technology discipline depends on turning data into useful information.

AI, cybersecurity, cloud operations, business analytics, and help desk reporting all depend on accurate interpretation.

## Interview-Ready Answer

Data is raw facts. Information is data that has been processed or placed in context so it can support understanding or decisions.

## Leadership-Ready Answer

Organizations create value when raw data becomes trustworthy information that supports decisions, operations, risk management, and strategy.

## How It Connects To Other Technology

- AI
- Cybersecurity
- Analytics
- Databases
- Reporting
- Governance
- Decision-making

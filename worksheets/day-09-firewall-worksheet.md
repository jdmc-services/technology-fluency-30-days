# Firewall Worksheet

## 1. What Is It?

A firewall is a control point that reviews network traffic and allows or blocks it based on rules.

## 2. Why Does It Matter?

Firewalls matter because they help reduce unauthorized access between systems, networks, and the internet.

## 3. Where It Shows Up At Work

Firewalls protect boundaries between users, applications, servers, cloud environments, vendors, and external networks.

## 4. How It Connects To AI

AI tools and APIs may require firewall rules, network restrictions, secure endpoints, and controlled data access paths.

## 5. How It Connects To Cybersecurity

Firewalls support segmentation, access control, threat reduction, monitoring, and incident containment.

## 6. How It Connects To Infrastructure

Infrastructure teams manage firewall rules, network zones, routing paths, VPN access, and application connectivity.

## 7. How It Connects To Business

Poor firewall governance can create outages, excessive access, audit findings, or security exposure.

## 8. Interview-Ready Answer

A firewall filters network traffic based on security rules to help protect systems from unauthorized access.

## 9. Leadership-Ready Answer

Firewalls enforce boundaries between systems, but they require governance so access does not become overly broad over time.

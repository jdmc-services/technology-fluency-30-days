# Cloud Computing Worksheet

## 1. What Is It?

Cloud computing means using computing resources delivered over a network, usually from a provider.

## 2. Why Does It Matter?

Cloud matters because it allows organizations to access servers, storage, databases, applications, security tools, and development platforms without building everything themselves.

## 3. Where It Shows Up At Work

Cloud supports email, file storage, video meetings, business applications, AI tools, databases, backup, analytics, and collaboration.

## 4. How It Connects To AI

Many AI tools depend on cloud infrastructure for model hosting, data processing, APIs, storage, and scaling.

## 5. How It Connects To Cybersecurity

Cloud security depends on identity, access control, encryption, logging, monitoring, configuration management, and shared responsibility.

## 6. How It Connects To Infrastructure

Infrastructure teams manage hybrid connectivity, identity integration, networking, monitoring, resilience, and cost control.

## 7. How It Connects To Business

Cloud can improve agility and scalability, but it requires governance, cost management, security, and resilience planning.

## 8. Interview-Ready Answer

Cloud computing provides on-demand access to computing resources such as servers, storage, and applications through a provider.

## 9. Leadership-Ready Answer

Cloud can improve agility and scalability, but it still requires cost control, security governance, identity management, and resilience planning.

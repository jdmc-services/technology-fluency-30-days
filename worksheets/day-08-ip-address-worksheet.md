# IP Address Worksheet

## 1. What Is It?

An IP address identifies a device on a network so traffic can be sent to and from that device.

## 2. Why Does It Matter?

IP addresses matter because devices need addresses to communicate across networks.

## 3. Where It Shows Up At Work

When users cannot connect, technicians often check IP address, gateway, DNS, subnet, and network settings.

## 4. How It Connects To AI

AI services, APIs, cloud resources, and model endpoints still rely on network addressing and routing.

## 5. How It Connects To Cybersecurity

IP addresses help security teams investigate traffic, logs, access patterns, suspicious sources, and network segmentation.

## 6. How It Connects To Infrastructure

Infrastructure teams manage IP addressing for devices, servers, printers, cloud systems, network equipment, and applications.

## 7. How It Connects To Business

Reliable IP addressing supports communication, application access, device management, and service availability.

## 8. Interview-Ready Answer

An IP address identifies a device on a network and helps route traffic to and from that device.

## 9. Leadership-Ready Answer

IP addressing is part of the foundation that allows users, devices, applications, and business systems to communicate reliably.

## 10. Practice Exercise

Explain the difference between a name and an address on a network.

## 11. Reflection Question

# Technology Fluency Worksheets: Days 1-10

These worksheets support the Technology Fluency: 30 Days learning path.

Each worksheet helps learners move from definition to application.

## Included Worksheets

- Day 1: DNS
- Day 2: MFA
- Day 3: AI Hallucination
- Day 4: Data vs. Information
- Day 5: API
- Day 6: Encryption
- Day 7: Help Desk Ticket
- Day 8: IP Address
- Day 9: Firewall
- Day 10: Cloud Computing

## Worksheet Structure

Each worksheet includes:

- What the concept means
- Why it matters
- Where it appears at work
- How it connects to AI
- How it connects to cybersecurity
- How it connects to infrastructure
- How it connects to business
- Interview-ready answer
- Leadership-ready answer
- Practice exercise
- Reflection question

## Purpose

The goal is to help learners understand the concept, explain it clearly, and apply it in real work.

# Day 6: Encryption

## Simple Explanation

Encryption converts readable data into an unreadable format so only authorized people or systems can access it.

It is like locking important information in a digital safe.

## Why It Matters At Work

Encryption helps protect personal information, financial data, business records, communications, and sensitive data while stored or transmitted.

## Interview-Ready Answer

Encryption protects data by converting it into an unreadable format that can only be restored with the proper key.

## Leadership-Ready Answer

Encryption is a foundational trust control that helps protect sensitive data, reduce exposure, and support security and compliance requirements.

## How It Connects To Other Technology

- Cybersecurity
- Cloud computing
- Data protection
- AI platforms
- Healthcare
- Finance
- Secure communications

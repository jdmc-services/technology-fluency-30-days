# Day 8: IP Address

## Simple Explanation

An IP address is a number assigned to a device so it can communicate on a network.

It works like a network address.

If data needs to reach your laptop, printer, server, or phone, the network needs to know where to send it.

## Why It Matters At Work

When a device cannot connect, technicians often check whether it has the correct IP address, gateway, DNS, and network settings.

## Interview-Ready Answer

An IP address identifies a device on a network and helps route traffic to and from that device.

## Leadership-Ready Answer

IP addressing is part of the foundation that allows users, devices, applications, and business systems to communicate reliably.

## How It Connects To Other Technology

- Networking
- DNS
- DHCP
- Cloud
- Cybersecurity
- Firewalls
- Troubleshooting
- Infrastructure

# Day 2: MFA

## Simple Explanation

MFA stands for Multi-Factor Authentication.

It requires more than one proof before someone can access an account.

A password is one proof. A phone prompt, authentication app, security key, certificate, or biometric check can be another proof.

## Why It Matters At Work

Passwords can be stolen, reused, guessed, or phished.

MFA makes it harder for a stolen password to become a successful login.

## Interview-Ready Answer

MFA strengthens authentication by requiring multiple factors, such as something you know and something you have.

## Leadership-Ready Answer

MFA reduces account takeover risk and helps protect critical systems when passwords are compromised.

## How It Connects To Other Technology

- Identity and Access Management
- Cloud security
- Remote access
- VPN
- Email security
- Privileged access
- AI platform access
- Cybersecurity governance

## Common MFA Types

- TOTP authentication app
- Push notification
- Number matching
- Hardware security key
- Certificate-based authentication
- Biometrics

## CIA Triad Connection

- Confidentiality: helps protect sensitive data from unauthorized access.
- Integrity: helps reduce unauthorized changes to systems or records.
- Availability: helps prevent account takeover from disrupting services.

# Day 9: Firewall

## Simple Explanation

A firewall is a control point that reviews network traffic and decides what should be allowed or blocked based on rules.

Those rules may consider source, destination, port, protocol, application, or user identity.

## Why It Matters At Work

Firewalls help reduce unauthorized access between networks, systems, and the internet.

A firewall is only as effective as the rules, review process, and monitoring behind it.

## Interview-Ready Answer

A firewall filters network traffic based on security rules to help protect systems from unauthorized access.

## Leadership-Ready Answer

Firewalls enforce boundaries between systems, but they require governance so access does not become overly broad over time.

## How It Connects To Other Technology

- Cybersecurity
- Network segmentation
- Zero Trust
- Cloud security
- Remote access
- Monitoring
- Incident response
